## 🚀 Getting Started

### Prerequisites

- Node.js 16+
- Git

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/knnthgms/neon-swamp.git
   cd neon-swamp
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start development server**

   ```bash
   npm start
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

The application will be available at `http://localhost:3000`
